# monitor_subdomains
Monitor your target continuously for new subdomains using Discord!


#Usage

-d => specify the domain name

-w => Discord Webhook

-t => Time interval after each scanning (seconds)


Example:

python3 monitor.py -d bepractical.tech -t 3600 -w <discord_webhook>
